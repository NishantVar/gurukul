# Welcome

**You can create anything you can imagine.** Intelligence has become so capable that the bottleneck is no longer technical skill — it's clarity of thought. If you can describe what you want, you can build it.

**Code is no longer the interface. Logic is.** You don't need to memorize syntax, master frameworks, or learn to type commands. You need to think clearly, communicate precisely, and verify results. The agent handles the rest.

**This isn't optional — it's survival.** A tool this powerful reshapes every field it touches. Learning to use it well isn't a nice-to-have. It's how you stay capable in a world that's moving fast.

## How to learn here

The curriculum is organized into tracks that build on each other. Start with the first lesson in your track and work through them in order. If you're starting from a later track, *skim the ones before it* — you don't need to do every exercise, but the mental models compound.

**When something doesn't make sense, ask the agent why.** That's not a workaround — it's the method. Every time the agent does something you don't understand, ask it to explain. That single habit is how you scale from building small things to building ambitious ones.

You don't need a course for every concept. You need the habit of asking.

**If something's off — or you have an idea — comment on the forum topic.** Every lesson has a forum thread. Feedback posted there gets automatically consolidated and folded back into the curriculum. You don't need to file a report or wait for someone to notice. Just say what's wrong or what's missing, and the material evolves.
